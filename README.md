## INSTRUCTION

> clone this repo by using git. Change directory, make executable and run the script
```bash
git clone --depth 1 https://github.com/AXWTV/Hyprland-Installer.git
cd Hyprland-Installer
chmod +x install.sh
./install.sh
```
